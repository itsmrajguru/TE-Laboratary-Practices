-------------------------------Line: 1, Column: 16 Method does not exist or incorrect signature: void createStudent() from the type StudentManager




Perfect — that means the class itself is fine.
The problem is almost certainly the field API name.

Line 8 is probably this:

s.Student_Name__c = 'Mangesh';

Salesforce is saying:

“This field name does not exist.”

Because Salesforce may have created a different API name automatically.

FIX IT

Go here carefully:

Setup → Object Manager → Student → Fields & Relationships

Now click the field:

Student Name

You will see something like:

Label	Student Name
Field Name	Student_Name

OR maybe:

Student_Name1
Student_Name_2

etc.

IMPORTANT

You need the:

API Name

If API Name is:

Student_Name

then code becomes:

s.Student_Name__c = 'Mangesh';

But if API Name is:

Student_Name1

then code must be:

s.Student_Name1__c = 'Mangesh';
Do This For ALL FIELDS

Check exact API names for:

Student Name
Roll Number
Department
Marks

Then replace them in code.
